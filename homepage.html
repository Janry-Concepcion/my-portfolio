Capstone Project – Centralized Record Management System

A web-based system designed to digitize barangay operations—complaint records, certificate issuance, residents’ information, demographic insights, and document scanners.

The design emphasizes clarity, simple navigation, readable typography, and efficient layout structure based on the needs of the Lupong Tagapamayapa, Barangay Secretary, Barangay Captain, and Certificate Department.

Water Refilling Station App

A mobile ordering and delivery app for clean drinking water. This project is a user-friendly mobile application designed for a local water refilling station. The app allows customers to conveniently place orders for refilled water containers and schedule deliveries directly from their phones.

The interface focuses on simplicity—users can choose container types, set quantities, and select delivery times with just a few taps. Real-time order status updates and delivery tracking improve customer experience, while the system also supports customer profiles and order history for seamless repeat ordering.

Role: System Designer / UI-UX Designer

Year: 2024-2025
